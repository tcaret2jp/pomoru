# デザイン仕様書

## 1. デザインコンセプト
「静寂と集中」。余計な要素を削ぎ落とし、タイマーそのものに没頭できるインターフェースを目指す。

## 2. カラーシステム
| 用途 | ダークモード (Default) | ライトモード |
| :--- | :--- | :--- |
| 背景 | `#000000` (Pure Black) | `#F9F9F9` (Off White) |
| テキスト (主) | `#FFFFFF` | `#1A1A1A` |
| テキスト (副) | `#A1A1AA` (Zinc 400) | `#71717A` (Zinc 500) |
| 作業中アクセント | `#EF4444` (Red 500) | `#DC2626` (Red 600) |
| 休憩中アクセント | `#10B981` (Emerald 500) | `#059669` (Emerald 600) |

## 3. タイポグラフィ
- **メインフォント**: Inter, system-ui, -apple-system (モダンで読みやすいサンセリフ体)
- **タイマー数字**: 等幅フォント（JetBrains Mono等）を使用し、カウントダウン中の数字の揺れを防ぐ。

## 4. コンポーネント
- **TimerDisplay**: 画面中央。大きく太いフォント。
- **ControlButtons**: タイマーの下に配置。アイコンとテキストの組み合わせ。
- **ModeSwitcher**: 「Work」「Short Break」「Long Break」を切り替えるタブ状のUI。
- **SettingsModal**: 右上のギアアイコンからアクセス。各時間の数値を入力。

## 5. アニメーション
- タイマーの開始/停止時のフェードイン・アウト。
- プログレスバー（タイマーの周囲に円形、または上部に直線状）の滑らかな動き。
